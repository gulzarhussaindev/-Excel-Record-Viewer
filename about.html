About Excel Record Viewer

Excel Record Viewer is built to let you open, edit and export spreadsheet files entirely in your browser — quickly, privately, and offline when installed as a PWA.

What this tool does

  • View spreadsheets (XLSX, XLS, CSV) without installing software.
  • Edit inline in table view — click any visible cell to update it.
  • Add new records using a simple form view that shows only the columns you selected.
  • Export updated files to download a new Excel/CSV with all your changes (original upload remains unchanged).
  • Customize visible columns with the Columns selector so forms and tables show only fields you care about.
  • Privacy-first — everything runs client-side, your files never leave your device.

Technical highlights

  • Uses a client-side XLSX parser (no uploads).
  • Form view for adding records — only shows selected columns.
  • Table view supports inline editing and sorting.
  • Export filename is friendly (ExcelViewer_Export_YYYY-MM-DD.xlsx).
  • PWA ready — install and use offline for basic viewing/editing workflows.
